## 📋 Guidelines

- Capture clear and high-quality screenshots.
- Record videos only when they help demonstrate the issue more effectively.
- Ensure the entire bug reproduction process is visible.
- Remove or blur any sensitive or personal information before uploading.
- Use descriptive file names that match the corresponding Bug ID.
